· Unicode: f147 · Created: v3.1 · Categories: Web Application Icons, Form Control Icons

After you get up and running, you can place Font Awesome icons just about anywhere with the tag:

fa-minus-square-o

<i class="fa fa-minus-square-o" aria-hidden="true"></i>

Note: to improve web accessibility, we recommend using aria-hidden="true" to hide icons used purely for decoration.
